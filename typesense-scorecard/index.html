Bedømmelsesmetode

  • 200-query benchmark: Algolias top-200 queries + GSC-queries. Top-1 resultat bedømt af heuristisk judge (brand/kategori/ordmatch + manuelle overrides).
  • Struktureret test: 182 håndplukkede queries i 7 buckets (symptomer, stavefejl, lange sætninger, ingredienser, brands, niche, koncepter).
  • Farvekodning: GREEN = perfekt match, YELLOW = relateret men ikke perfekt, RED = irrelevant, ZERO = ingen resultater.
  • Live søgning: Kalder Typesense API direkte (search-only key) + Med24.dk søgeside for sammenligning.

Konklusion

  • Typesense (95% GREEN) og Med24 nuv. (94% GREEN) er tæt på lige på standard queries
  • Begge er markant bedre end Matas (78%)
  • Typesense er 50-100x hurtigere (3ms vs 200-300ms)
  • Stavefejl (94%) slår Algolia (81%). Symptomer (73%), koncepter (75%), lange sætninger (75%) kraftigt forbedret
  • Styrker: Stavefejl (94%), ingredienser (90%), brands (83%), niche/edge cases (80%)
