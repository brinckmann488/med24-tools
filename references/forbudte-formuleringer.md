# Forbudte formuleringer i kosttilskudsmarkedsføring

## Sygdomsanprisninger (ALTID FORBUDT)

Enhver påstand om forebyggelse, behandling eller helbredelse af sygdom er forbudt.

### Eksempler på forbudte sygdomsanprisninger

#### Direkte sygdomsreferencer
- "forebygger kræft"
- "behandler diabetes"
- "helbreder gigt"
- "mod forhøjet blodtryk"
- "lindrer depression"
- "kurerer migræne"
- "modvirker Alzheimers"
- "hjælper mod astma"

#### Symptomreferencer der antyder sygdom
- "lindrer ledsmerter" → antyder gigt/artrose
- "sænker blodsukkeret" → antyder diabetes
- "normaliserer blodtrykket" → antyder hypertension
- "letter vejrtrækningen" → antyder astma/KOL
- "fjerner inflammation" → antyder inflammatorisk sygdom
- "modvirker angstanfald" → antyder angstlidelse
- "hjælper mod søvnløshed" → kan antyde søvnforstyrrelse

#### Medicinske termer
- "terapeutisk"
- "klinisk bevist"
- "lægeanbefalet til behandling af..."
- "medicinsk effekt"
- "farmakologisk virkning"

### Grænsetilfælde - ofte problematiske

| Formulering | Problem | Muligt alternativ |
|-------------|---------|-------------------|
| "styrker immunforsvaret" | For generel | "bidrager til normal funktion af immunsystemet" (hvis godkendt) |
| "giver energi" | Uspecifik | "bidrager til normal energiomsætning" (hvis godkendt) |
| "godt for hjertet" | Uspecifik sundhedsanprisning | Brug godkendt anprisning for EPA/DHA |
| "anti-aging" | Antyder forebyggelse af aldring som sygdom | Undgå |
| "detox" | Antyder medicinsk rensning | Undgå |

## Ikke-godkendte sundhedsanprisninger

Sundhedsanprisninger der IKKE er på EU's godkendte liste:

### Almindelige afviste claims
- "styrker knoglerne" (brug i stedet: "bidrager til vedligeholdelse af normale knogler")
- "forbedrer hukommelsen" (ingen generel godkendelse)
- "øger forbrændingen" (ikke godkendt)
- "styrker håret" (brug: "bidrager til vedligeholdelse af normalt hår" for zink/biotin)
- "giver bedre søvn" (ikke godkendt generelt)
- "reducerer stress" (ikke godkendt)
- "booster metabolismen" (ikke godkendt)

### Plantebaserede claims "on hold"
Mange planteanprisninger er "on hold" i EU siden 2012 — de er hverken godkendte eller afviste.
On-hold claims klassificeres som `NEEDS_REVIEW` med severity `LOW` og må stadig bruges i markedsføring, så længe de ikke er vildledende.

Eksempler på on-hold claims:
- Ginkgo biloba og hukommelse
- Valeriane og søvn
- Echinacea og immunforsvar
- Kurkuma og ledfunktion
- Guarana og energi (bemærk: koffein-specifikke claims er godkendte)

Se `references/botanicals-on-hold.md` for den fulde oversigt og klassifikationsregler.

## Vildledende formuleringer

### Overdrivelser
- "mirakel-", "vidunder-", "super-"
- "revolutionerende", "banebrydende"
- "100% effektivt"
- "garanteret resultat"

### Falsk autoritet
- "Læger anbefaler" (uden dokumentation)
- "Videnskabeligt bevist" (uden reference)
- "Kliniske studier viser" (uden specifikation)

### Vildledende sammenligninger
- "Bedre end medicin"
- "Naturligt alternativ til [lægemiddel]"
- "Samme effekt som [lægemiddel]"

### Testimonials der antyder effekt
- "Jeg blev kureret for min gigt"
- "Mit blodsukker er nu normalt"
- "Jeg har ikke haft migræne siden"

## Fødevarestyrelsens fokusområder (2024-kontrol)

Ifølge seneste kontrolrapport var de hyppigste overtrædelser:

1. **Uspecifikke sundhedsanprisninger** - vage påstande uden godkendelse
2. **Sygdomsanprisninger** - påstande om lindring/forebyggelse
3. **Manglende kobling** - uspecifikke claims uden reference til godkendt anprisning

## Sikre formuleringer

Neutralt og lovligt:
- "Indeholder vitamin C" (faktuel)
- "Med zink og selen" (ingrediensliste)
- "Fødevarestyrelsen anbefaler D-vitamin til..." (officiel anbefaling)
- "Til voksne" (målgruppe uden sundhedsclaim)
