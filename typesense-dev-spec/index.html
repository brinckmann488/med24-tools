1Overblik

Variant A — Nuvaerende (Algolia)

  • Endpoint: POST /searchSuggestions
  • Svartid: 200-300 ms
  • Returnerer: flat liste (forslag + produkter blandet)
  • Kvalitet: 89% GREEN (200 benchmark queries)

Variant B — Ny (Typesense)

  • Endpoint: POST /multi_search
  • Svartid: 4 ms (50-100x hurtigere)
  • Returnerer: 3 strukturerede sektioner
  • Kvalitet: 91% GREEN (200 benchmark queries)
Visuelt layout (A2 — Med Genveje): Venstre sidebar med MAERKER (top) + FORSLAG (midt) + GENVEJE (bund). Hoejre side med PRODUKTER (top 5, brand-diversificeret). Genveje er brand+kategori kombinationer som klikbare chips (fx "Derma solcreme", "CeraVe solcreme"). Se reference-implementation.

4MAERKER — extractBrands()

Unikke parent-brands fra top-30 produkthits. Max 6. Query-match sorteres oeverst.

function extractBrands(hits, q, limit = 6) {
  const seen = new Set();
  const brands = [];
  for (const h of hits) {
    const bp = h.document.brand_parent || h.document.brand;
    if (!bp || seen.has(bp.toLowerCase())) continue;
    seen.add(bp.toLowerCase());
    brands.push(bp);
  }
  const ql = (q || '').toLowerCase();
  brands.sort((a, b) => {
    const aMatch = a.toLowerCase().includes(ql) || ql.includes(a.toLowerCase());
    const bMatch = b.toLowerCase().includes(ql) || ql.includes(b.toLowerCase());
    return bMatch - aMatch;
  });
  return brands.slice(0, limit);
}
  • Datakilde: results[0].hits
  • Dedup via brand_parent ("Solaray Magnesium Citrat" og "Solaray" = en brand)
  • Ved klik: fyld soegefeltet med brandnavnet

Klikbare genveje der kombinerer brand + kategori. Vises som chips/pills under FORSLAG i sidebar.

const shortcuts = results[1].hits

  • Datakilde: results[1].hits — samme som FORSLAG, men filtreret paa context === 'brand'

6PRODUKTER — diversifyHits()

Vis 5 produkter fra de 30 hits, med brand-diversitet:

function diversifyHits(hits, q, limit = 5) {
  if (!hits || hits.length <= limit) return hits.slice(0, limit);

  // Brand intent check: if query contains a brand name, skip diversity
  const ql = (q || '').toLowerCase();
  const topBrands = [...new Set(hits.slice(0, 10).map(h =>
    h.document.brand_parent || h.document.brand).filter(Boolean))];
  const hasBrandIntent = topBrands.some(b =>
    ql.includes(b.toLowerCase()) || b.toLowerCase().includes(ql));
  if (hasBrandIntent) return hits.slice(0, limit);

  // Broad search: max 2 products per brand_parent
  const brandCount = {};
  const diverse = [], overflow = [];
  for (const h of hits) {
    const bp = (h.document.brand_parent || h.document.brand || '').toLowerCase();
    brandCount[bp] = (brandCount[bp] || 0) + 1;
    (brandCount[bp] <= 2 ? diverse : overflow).push(h);
  }
  return [...diverse, ...overflow].slice(0, limit);
}
Brand intent: Naar brugeren soeger "derma solcreme" vises kun Derma-produkter (ingen diversitet). Naar brugeren soeger "solcreme" vises max 2 produkter per brand.

10A/B test implementation

Cookie-baseret split, sticky per bruger i 30 dage:

// Assign variant (once per user)
function getSearchVariant() {
  const match = document.cookie.match(/search_ab=(ts|algolia)/);
  if (match) return match[1];
  const variant = Math.random() < 0.5 ? 'ts' : 'algolia';
  document.cookie = `search_ab=${variant};path=/;max-age=${30 * 86400}`;
  return variant;
}

const variant = getSearchVariant();

// Route search based on variant
async function search(query) {
  if (variant === 'ts') {
    return searchTypesense(query);  // POST /multi_search
  } else {
    return searchAlgolia(query);    // POST /searchSuggestions (existing)
  }
}

Maaling

  • Primary metric: Klik-through rate paa soegeresultater
  • Secondary: Soeg-til-koeb konvertering, bounce rate, tid til foerste klik
  • Send search_ab variant som GA4 user property for analyse
