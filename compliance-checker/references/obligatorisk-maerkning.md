# Obligatorisk mærkning af kosttilskud

## Påkrævede elementer på alle kosttilskud

### 1. Varebetegnelse
- Skal være: **"Kosttilskud"**
- Skal fremgå tydeligt

### 2. Kategoribetegnelse
- Navn på næringsstoffer/stoffer der kendetegner produktet
- Fx "Kosttilskud med vitamin D" eller "Kosttilskud med omega-3 fedtsyrer"

### 3. Ingrediensliste
- Alle ingredienser i faldende orden efter vægt
- Tilsætningsstoffer med E-nummer eller navn + funktion
- Planteingredienser: dansk navn + latinsk navn

### 4. Næringsdeklaration for kosttilskud
- Mængde af vitaminer/mineraler/stoffer per anbefalet daglig dosis
- Vitaminer og mineraler: også i % af referenceværdi (RI/NRV)

### 5. Anbefalet daglig dosis
- Klar angivelse af hvor meget der skal indtages
- Fx "1 tablet dagligt" eller "2 kapsler om morgenen"

### 6. Advarsler (obligatoriske)
- **"Overskrid ikke den anbefalede daglige dosis"**
- **"Bør ikke træde i stedet for en varieret kost"**
- **"Opbevares utilgængeligt for små børn"**

### 7. Nettoindhold
- Antal tabletter/kapsler ELLER vægt/volumen
- Fx "60 tabletter" eller "100 ml"

### 8. Holdbarhed
- "Mindst holdbar til: [dato]" eller "Mindst holdbar til: se [placering]"

### 9. Opbevaringsanvisning
- Hvis særlige betingelser kræves
- Fx "Opbevares køligt og tørt"

### 10. Virksomhedsoplysninger
- Navn/firmanavn på ansvarlig fødevarevirksomhed
- Adresse i EU

### 11. Oprindelsesland
- Påkrævet hvis udeladelse kan vildlede

## Særlige krav ved sundhedsanprisninger

Når der bruges godkendte sundhedsanprisninger, skal følgende også fremgå:

1. **Erklæring om varieret kost**: "En varieret og afbalanceret kost og en sund livsstil er vigtig"

2. **Mængde og måde**: Information om hvor meget og hvordan produktet skal indtages for at opnå den anpriste effekt

3. **Advarsler til risikogrupper**: Hvis relevant - fx gravide, børn, personer med visse sygdomme

4. **Advarsel mod overdosering**: Hvis relevant for det specifikke næringsstof

## Særligt krav for koffein

Kosttilskud med koffein skal mærkes med:

> **"Indeholder koffein. Bør ikke indtages af børn eller gravide (Xmg koffein pr. anbefalet daglig dosis)"**

Skal stå i samme synsfelt som varebetegnelsen.

## Sprogkrav

- Mærkning skal være på **dansk, svensk eller norsk**
- Let læselig skrift
- Synlig og ikke skjult

## Tjekliste for mærkningskontrol

```
[ ] Varebetegnelsen "Kosttilskud" er angivet
[ ] Ingrediensliste er komplet og korrekt
[ ] Planter har både dansk og latinsk navn
[ ] Næringsdeklaration med mængder per daglig dosis
[ ] Vitaminer/mineraler har %-RI angivet
[ ] Anbefalet daglig dosis er angivet
[ ] Advarsel om ikke at overskride dosis
[ ] Angivelse om at ikke erstatte varieret kost
[ ] Opbevaringsanvisning for børn
[ ] Nettoindhold angivet
[ ] Holdbarhedsdato angivet
[ ] Virksomhedsoplysninger komplet
[ ] Mærkning på dansk/svensk/norsk
[ ] Koffein-advarsel (hvis relevant)
[ ] Pligttekster ved sundhedsanprisninger (hvis relevant)
```

## Hyppige mangler

Fra Fødevarestyrelsens kontroller:

1. Manglende latinsk navn på planteingredienser
2. Ufuldstændig næringsdeklaration
3. Manglende %-RI for vitaminer/mineraler
4. Uklar dosering
5. Manglende obligatoriske advarsler
6. Manglende pligttekster ved sundhedsanprisninger
